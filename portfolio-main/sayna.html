    👥 Équipe

    4 personnes

    🛠️ Technologies

    Unity, C#

    📖 Description

    SAYNA est un projet mobile de jeu de stratégie au tour par tour éducatif basé sur un univers de science fiction. Le but de ce jeu est d'apporter un complément ludique à la plateforme éducative SAYNA, une plateforme en ligne pour apprendre la programmation. Le je n'a pas pu être terminé pendant ma période chez SAYNA.

    🎯 Mon rôle

    - programmeur UI

    ⚡ Challenges

    Le challange sur ce projet à été d'intégrer et de programmer l'UI et surtout de respecter les différentes animations de l'équipe UI/UX Design.

    🎯 Problème : Intégration et programmation d'une UI complexe

    Les jeux stratégiques ont souvent des UI riches et complexes ce qui m'a permis de d'explorer et de mieux comprendre la programmation UI sur Unity.

    🎯 Problème : Intégrer et programmer l'UI

    Comment programmer l'Ui pour paramètrer la simulation (nombre et type d'entités ennemis, type d'armement, etc...)?

    👉 Solution : Intégrer un systeme de capture de transform d'objets (Position,rotation, etc...) à chaque frame. Pour le replay le systême rejoue par interpolation les datas capturés pour reconstruire l'action de la scène.
