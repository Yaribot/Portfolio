Abstract Dream

Gameplay Programming / UI Programming

Duration

1 week

Team

1 person

Technologies

Unity, C#

Description

Abstract Dream is a small project focused on level design. The goal of the project was to explore perspective and use it to mislead the player. The isometric view can create situations where the player believes they can jump from one platform to another, only to realize that the jump is impossible because the target platform is actually farther away than it appears.


My role

- Gameplay Programmer
- UI Programmer


Challenges

The main challenge of this project was creating a shader that allows the player to be visible through walls.

Challenge: Seeing the player through walls

This shader, applied to environment objects, works as a mask based on the player’s position. It is triggered using a raycast from the camera to the player. If the ray hits a wall using the shader, the mask is activated, creating a cutout effect around the player’s position. This allows the player to remain visible even when obstructed by walls.

Problème : Intégrer et programmer l'UI

Comment programmer l'Ui pour paramètrer la simulation (nombre et type d'entités ennemis, type d'armement, etc...)?

➤ Solution : Intégrer un systeme de capture de transform d'objets (Position,rotation, etc...) à chaque frame. Pour le replay le systême rejoue par interpolation les datas capturés pour reconstruire l'action de la scène.
